# 清理项目文件

## 执行步骤

### 1. 扫描项目

扫描 `.require-agent/projects/` 下所有项目目录。

### 2. 识别可清理项目

读取每个项目的 state.json，筛选 phase = `completed` 的项目。

### 3. 展示清理计划

```
🧹 以下已完成项目可清理中间文件：

| 项目 | 版本数 | 评分文件数 | 预计释放 |
|------|--------|-----------|---------|
{每个已完成项目一行}

清理将保留：最终版本、state.json、进化日志
清理将删除：中间版本快照、中间评分记录

执行清理？(Y/N)
```

### 4. 执行清理

用户确认后：
- 保留 versions/ 中最终版本（版本号最大的）
- 删除其他中间版本
- 保留 scores/ 中 round-0.json 和 final.json
- 删除其他中间评分
- 输出清理结果
